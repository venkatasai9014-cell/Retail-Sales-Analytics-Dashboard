# Retail-Sales-Analytics-Dashboard
## Project Overview

This project analyzes e-commerce sales data using Python, SQL, and Power BI to generate business insights and interactive dashboards.

The project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- SQL Business Queries
- KPI Analysis
- Power BI Dashboard Development

---

# Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- SQL
- Power BI

---

# Dataset

The project uses the Superstore sales dataset containing:
- Orders
- Customers
- Products
- Sales
- Profit
- Shipping Information

---

# Python Analysis

Performed:
- Missing value handling
- Duplicate removal

- KPI calculations
- Trend analysis
- Data visualization

# SQL Analysis

Created SQL queries for:
- Sales Analysis

- Top Products

## 2. Product Analysis Dashboard

## 3. Customer & Shipping Dashboard
- Customer Analysis
- Segment Performance
- Shipping Analysis

---

# Key Business Insights

- Technology category generated highest profit.
- West region achieved highest sales.
- Consumer segment contributed maximum revenue.
- Standard shipping mode was most preferred.
